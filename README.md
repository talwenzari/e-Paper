# Personalized e-Paper Display Project
This is a fork of the Waveshare e-Paper repository, customized for displaying images from a file system location on a 7.5 inch e-paper display connected to a Raspberry Pi 4 via HAT interface.</br>
![waveshare_logo.png](waveshare_logo.png)

## Purpose

The main goal of this project is to create a Python program that monitors a specified directory for image files and automatically displays them on the e-paper display. This allows for dynamic content updates without manual intervention.

## Original Repository

This project is based on the [Waveshare e-Paper repository](https://github.com/waveshareteam/e-Paper), which provides libraries and examples for various e-paper displays across different platforms.

## Supported Platforms

* Raspberry Pi (Python and C examples)
* Jetson Nano
* Arduino
* STM32

## Getting Started

1. Set up your Raspberry Pi with the 7.5 inch e-paper HAT
2. Install the required dependencies
3. Run the image polling script to display images automatically
